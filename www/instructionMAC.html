Eagle for Mac OS

A How To guide for installing Eagle along with multi-threaded R

Purpose of this guide

Eagle is built with the R language. R, by default, is installed for single-thread computation. However, it is possible to replace R’s default single-thread math libraries with multi-threaded libraries. In doing so, many of R’s linear algebra operations become implicitly parallelized. We have purposely built Eagle to take full advantage of this implicit parallelisation.

In this guide, we will show you how to install R for multi-threaded computation. Once R is installed, it is then simple to install Eagle.

What you need

You will need administrator privileges. This gives the installation script write access to write-protected parts of the system.

It would also be advantageous if you have had previous experience with installing software to your computer and experience with installing R packages.

Systems Tested

We have tested our installation procedure on macOS 10.11 (El Capitan), and 10.12 (Sierra).

Step 1: Install XQuartz

Go to the XQuartz web site and download the dmg file. The file will be downloaded into your DOWNLOADS folder. Click on the XQuartz dmg file in your DOWNLOADS folder and a window will appear with the icon . Double click on the icon and the installation program will begin. You will see the following. You will need administrator privileges to install XQuartz.

Step 2: Download R Open

Installing a multi-threaded version of R is now much easier with Microsoft R Open. R Open behaves exactly the same as R, except, by default, it comes with multi-threaded math libraries.

Go to the Microsoft R Open web site and download the Mac OS X version of R Open.

Step 3: Run the installation program for R Open

Once R Open has finished downloading, run the program. There are various ways of doing this. Our preferred approach is to go into the Download folder and click on the file microsoft-r-open-3.4.0.pkg. This will run the installation program. You will need administrator privileges to run this program.

Step 4: Run R Open

R Open can be run as an App from the Launchpad

By clicking on the R Open App, the following R console window will appear. It tells you how many cores are available for distributed computation.

Step 5: Installation of the Eagle package

To install the Eagle package, from the R Console window, use the install.packages( ) command 

  install.packages("Eagle", dependencies=TRUE)

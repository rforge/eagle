Demo

Use our web-based GUI to analyse some test data

Try Eagle

If you would like to try out Eagle before installing the package, click the above button. This will take you to our demo server which is running the Eagle GUI. For a quick start guide on how to use our GUI, see here . For a video on how to use our GUI, see here .

Our demo server has three data files. These are a marker file (genoDemo.dat), a phenotypic file (phenoDemo.dat), and a optional map file (mapDemo.dat).

Here's what's in each file:

  • the genoDemo.dat file contains the genotypes from 4998 snp on 150 individuals. The file is plain text and space separated. The data are coded as 0, 1, and 2, corresponding to snp genotypes AA, AB, and BB, respectively. Click here if you would like to download the data.
  • the phenoDemo.dat file contains the phenotypic data from the 150 individuals. Data are recorded from two traits (trait1, trait2), and four predictors. It is a space separated text file with the first row containing the column headings. Click here if you would like to download the data.
  • the mapDemo.dat file contains the marker map for the 4998 snp. It is a space separated text file containing three columns (SNP, Chrm, Pos). The first row contains the column headings. Click here if you would like to download the data.

Try It Out Now

Use our web-based GUI to analyse some test data

Try Eagle

By clicking on the above button, you will be taken to the Eagle GUI where you can analyse some test data with Eagle. This is for demonstration purposes only. You are unable to upload data for analysis due to the limited resources available on our test server. The test data on our server is as follows:

  • the marker data are contained in the file genoDemo.dat. It is a plain text file, containing the values 0, 1, and 2, for the snp genotypes AA, AB, and BB, respectively.
  • the phenotypic data are contained in the file phenoDemo.dat. It is space separated, and the first row contains the column headings.
  • the map is contained in the file mapDemo.dat. It is a space separated file, and the first row contains the column headings.

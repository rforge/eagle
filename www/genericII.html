Eagle

Getting Started

To install the Eagle package, from an interactive session in R, type the following command 

install.packages("Eagle", dependencies=TRUE)
This will install the Eagle package and any missing packages upon which Eagle depends. As part of the install, you will be prompted to chose a mirror site from which to install the packages. Chose the site closest to your location.

The Eagle package is loaded into your R session with 

library("Eagle")
and you will see a message saying that Eagle has been loaded.

Running Eagle via the GUI

The easiest way of interacting with Eagle and performing an analysis is through our web-based graphical user interface (GUI). To run the GUI, at the R prompt, type 

OpenUI()
which will automatically open your web browser to our GUI.

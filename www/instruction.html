How to Install

Eagle

For those in a hurry

Eagle is built on the R language. If you just want to try Eagle out, run R (assuming you already have R installed) and from an interactive session, type the command 

install.packages("Eagle", dependencies=TRUE)
This will install the Eagle package and any missing packages upon which Eagle depends.

To get started analysing data with Eagle, see our Quick Start page for easy to follow instructions.

Super-charging Eagle with multi-thread computation

Eagle is built on the R language. When you install R, by default, R is installed with single-thread math libraries. However, it is possible to replace these single-thread libraries with multi-thread libraries. In doing so, many of Râ€™s linear algebra operations become implicitly parallelized. We have purposely built Eagle to take full advantage of these implicitly-parallelised linear algebra operations.
